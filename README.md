# Placement Management System

A backend-based Placement Management System developed using Java, Spring Boot, REST APIs, JDBC, SQL, and Postman.

## Features
- Add student details
- Update student information
- View all students
- Delete student records
- REST API integration
- CRUD operations

## Technologies Used
- Java
- Spring Boot
- REST APIs
- JDBC
- PostgreSQL
- Postman

## Testing
- API testing performed using Postman
- CRUD operations validated through request/response testing
- Backend workflows debugged and verified
